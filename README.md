# Nicemp Tools Platform

Production Vite + React + TypeScript app for the Nicemp tools ecosystem.

## Stack

- Vite
- React
- TypeScript
- TailwindCSS
- shadcn/ui
- Wouter
- Lucide React
- Recharts
- Supabase-ready service layer

## Environment

Copy `.env.example` to `.env.local` for local development and configure Supabase keys when authentication and persistence are enabled.

## Scripts

- `npm run dev`
- `npm run typecheck`
- `npm run build`
- `npm run serve`
